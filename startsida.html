World Wide Web (WWW) är ett system av sammankopplade dokument och resurser som nås via internet. Det skapades av Tim Berners-Lee 1989 och bygger på hypertext, vilket gör att användare kan navigera mellan olika webbsidor genom att klicka på länkar. Webben använder sig av HTTP-protokollet (HyperText Transfer Protocol) för att överföra information mellan klienter och servrar. Dokumenten på webben är oftast skrivna i HTML (HyperText Markup Language) och kan innehålla text, bilder, videor och andra typer av media. Kort sagt, World Wide Web är det system som gör att vi kan surfa på nätet och se innehåll som webbsidor.
